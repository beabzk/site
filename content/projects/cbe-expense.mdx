A React + Vite expense tracking application with modern UI and enhanced user experience. Features UI overhauls, dark mode support, improved UX, file input capabilities, and transaction extraction. Built with modern React patterns and optimized for performance with responsive design and intuitive user interface.

## Highlights
- **Modern UI/UX**: Completely overhauled interface with a focus on ease of use.
- **Theme Support**: Built-in dark mode for comfortable financial tracking.
- **Transaction Extraction**: Tools to help extract and categorize transactions efficiently.
- **Performance**: Optimized using Vite and modern React patterns.
